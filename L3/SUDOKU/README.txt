Il s'agit d'un projet de résolution de Sudoku a été mené en 3ème année de Licence par : 
	- Téo Itsweire-Krebs
	- Antoine Promis
	- Arnaud Bonnet
	- Julien Lefevre
A partir d'une grille partiellement remplie, ce projet à pour but de compléter autant que possible la grille à l'aide de règles de base de résolution des Sudoku. 

Pour compiler, il faudra saisir la ligne de commande suivante : 
- Windows : gcc main.c Grille.c Notes.c Resolution.c -o sudoku.exe
- Linux : gcc main.c Grille.c Notes.c Resolution.c -o sudoku -lm

Puis pour exécuter : 
- Windows : ./sudoku.exe
- Linux : ./sudoku
