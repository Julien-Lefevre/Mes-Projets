Ce projet de Base de données en 3ème année de Licence a été mené par : 
	- Téo Itsweire-Krebs
	- Arnaud Bonnet
	- Antoine Promis
	- Julien Lefevre
L'objectif était la création d'une base de données pour une plateforme de streaming nommée NetFlouze. Ainsi, il fallait premièrement créer nos tables avant de les remplir à l'aide de données à trier dans un fichier donné. 

Pour faire tourner ce projet vous pouvez : 
	- Utiliser une VM et avec Oracle et sqldeveloper, 
	- ou installer par exemple SQLite et l'utiliser dans vsCode.
